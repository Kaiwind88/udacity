computeValueFromQValues(self, state)

Returns max_action Q(state,action) where the max is over legal actions. Note that if there are no legal actions, which is the case at the terminal state, we return a value of 0.0.

Parameters:

Parameters:

getAction(self, state)

source code 

Compute the action to take in the current state. With probability self.epsilon, we take a random action and take the best policy action otherwise. Note that if there are no legal actions, which is the case at the terminal state, we choose None as the action.

Parameters:
  • state (tuple) - given tuple state (location) of robot

  • nextState (tuple) - next resulting state of robot after taking action
  • reward (float) - reward received from current state by taking action
