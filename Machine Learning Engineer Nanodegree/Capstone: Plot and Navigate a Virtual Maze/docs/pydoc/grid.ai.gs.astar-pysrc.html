 9      ''' 
10      A* is an informed search algorithm, or a best-first search, meaning that it solves  
11      problems by searching among all possible paths to the solution for the one that  
12      incurs the smallest cost (least distance travelled, shortest time, etc.), and among  
13      these paths it first considers the ones that appear to lead most quickly to the solution.  
14      It is formulated in terms of weighted graphs: starting from a specific node of a graph,  
15      it constructs a tree of paths starting from that node, expanding paths one step at a time,  
16      until one of its paths ends at the predetermined goal node. 
17       
18      At each iteration of its main loop, A* needs to determine which of its partial paths to  
19      expand into one or more longer paths. It does so based on an estimate of the cost (total weight)  
20      still to go to the goal node. Specifically, A* selects the path that minimizes 
21   
22      f(n) = g(n) + h(n) 
23   
24      where n is the last node on the path, g(n) is the cost of the path from the start node to n,  
25      and h(n) is a heuristic that estimates the cost of the cheapest path from n to the goal.  
26      The heuristic is problem-specific. For the algorithm to find the actual shortest path, the heuristic  
27      function must be admissible, meaning that it never overestimates the actual cost to get to the  
28      nearest goal node. Typical implementations of A* use a priority queue to perform the repeated  
29      selection of minimum (estimated) cost nodes to expand. 
30      '''
