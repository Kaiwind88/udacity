drawPatch(self, x, y, color='#E5E4E2', key_prefix='')

source code 

Patch is a gray square that signifies unexplored cell. When a cell is visited, we remove this patch and cell becomes white.

Parameters:
  • x (int) - x coord of cell
  • y (int) - y coord of cell
  • color (Hex color code) - color of patch. Default is a gray color
  • key_prefix (string) - prefix used to uniquely identify eraseable turtle to draw this patch
